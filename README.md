# OPC超级个体训练营｜UI优化上线版

本版本为静态 GitHub Pages 可上线版本，重点优化了：

- 登录页企业化改版
- 深色 SaaS 风格工作台
- 左侧导航与分级目录重做
- 培训营、物料资料、素材案例、其他资料统一为更清晰的资料系统
- 平台标识内置，不依赖外部图片资源
- 保留管理员上传、删除、注册申请查看等前端演示能力

## GitHub 上传要求

解压后，仓库根目录必须直接看到：

- `index.html`
- `styles.css`
- `app.js`
- `404.html`
- `.nojekyll`
- `README.md`

不要上传外层文件夹，不要只上传 ZIP。
